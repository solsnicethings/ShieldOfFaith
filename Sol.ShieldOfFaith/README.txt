-- Shield of Faith --

Screen dimming software by Sol, made available under GNU GPLv3 license.

Details available within the program.

Run Sol.ShieldOfFaith.exe to begin.

The program should be bundled with ShaderGlassForShieldOfFaith.exe, but even without ShaderGlass, it has useful functionality.

It is a version of ShaderGlass.exe that Shield of Faith can start, which will then respond to signals sent by Shield of Faith, allowing you to control the program without directly accessing its own user interface.

https://github.com/solsnicethings/ShieldOfFaith

-- Shader Glass --

For a very capable user, ShaderGlass might be sufficient in and of itself, but it is a gaming-oriented project, not an accessibility oriented one.

The user interface of ShaderGlass is not designed with the same use case in mind as Shield of Faith is.

https://mausimus.github.io/ShaderGlass/MANUAL.html

-- Shader Glass for Shield of Faith --

Via Shield of Faith, you get a darker user interface, and can access property sliders directly from a controlling window, instead of having to access a menu system within the monitor-dimming window itself.

For a user whose purpose is to limit exposure to certain visual phenomenons (flashing, brightness and such), this is intended to be a more suitable experience. Also, Shield of Faith provides ShaderGlass with custom shaders (.slang files) that are specifically designed to give control over the light from the monitor.
